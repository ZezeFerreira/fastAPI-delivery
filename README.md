🍕 FastAPI Pizza Delivery API
REST API para gerenciamento de pedidos de uma pizzaria, desenvolvida com FastAPI, SQLAlchemy e autenticação JWT.

Tecnologias
Python
FastAPI
SQLAlchemy
SQLite
JWT (JSON Web Token)
Passlib (bcrypt)
Alembic
Funcionalidades
= Cadastro de usuários

Autenticação com JWT (access + refresh token)
Criação de pedidos
Adição e remoção de itens no pedido
Cálculo automático do valor total
Cancelamento e finalização de pedidos
Controle de acesso (usuário e administrador)
